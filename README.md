A UI Library for Vue.
## LUI

### Download
```sh
    git clone https://github.com/zeqi/l-ui.git

```
### Iteration plan
```sh
    1.Lbtton
    2.Lmodal
    3.Lprogress
    4.Licon
    5.Lswitch
    6.Ltabs
    7.Lcheckout
    8.Lline
    9.Lspin
    10.Lselect
    11.Lradio
    12.Lslider
```
### Usage
```html
    <template>
      <Lbtton type='primary' />
    </template>
```
### Compatibility
```sh
  * Supports Vue.js 2.x
  * Supports Vue.js 1.x 
```
